## Projets

### Projet 1 (Java):Application de gestions de motos
Objectif du projet:  
Ce projet a été réalisé pour pratiquer la programmation orientée objet en Java ainsi que la structuration d’une application en plusieurs couches (modèle, service, interface).  
Fonctionnalités  
L’application permet de :  
      - Ajouter une moto  
      - Afficher la liste des motos  
      - Supprimer une moto  
      - Rechercher une moto  
Toutes les interactions se font via un menu dans la console.  

Technologies utilisées  
      - Java  
      - Programmation orientée objet  
      - Architecture simple (Model / Service / Main)  
      - Structure du projet  

Moto : classe représentant une moto (marque, modèle, année, kilométrage, prix)  
MotoService : logique de gestion des motos  
Main : interface console et menu utilisateur  
Objectif pédagogique  
Ce projet fait partie de mon apprentissage en développement logiciel. Il m’a permis de comprendre :  
      - la gestion d’objets en Java  
      - la séparation entre logique métier et interface  
      - la création d’une application console interactive  

Évolution du projet  
Une future version du projet pourrait inclure :  
      - une interface graphique avec JavaFX  
      - la sauvegarde des données dans un fichier ou une base de données
